# Modules Go

Ce dossier accueille les projets écrits en Go.
Chaque sous-répertoire doit contenir son propre `go.mod`.

## Construire et lancer
```bash
go build
./<nom_du_binaire>
```

## Tests
```bash
go test ./...
```

Décrivez les packages ou commandes disponibles pour chaque module.
